# sistem-penjualan-tanaman
Sistem Informasi Penjualan Tanaman berbasis website yang dikembangkan untuk membantu CV Purnama Sejahtera dalam mengelola data produk, pemesanan, transaksi, dan laporan penjualan. Sistem ini dibangun menggunakan Laravel dan MySQL dengan antarmuka yang memudahkan admin mengelola data serta pelanggan melakukan pembelian secara online.
